## PyRat
 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;PyRat is a maze game with two opponents (a rat and a python), some pieces of cheese, and potentially obstacles and mud. The  goal is to collect as much pieces as possible.
